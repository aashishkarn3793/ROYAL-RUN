# Royal Run 👑

A fast-paced 3D endless runner built with Unity where players control a royal character, dodge obstacles, collect coins, and survive as long as possible while racing through an endless castle pathway.

---

## 🎮 Overview

Royal Run is a reflex-based endless runner focused on precision movement, obstacle avoidance, and high-score progression.

Players must navigate an endless royal bridge, avoid incoming obstacles, collect coins, and survive for as long as possible to set new high scores.

The challenge increases the longer you survive, testing reaction speed and focus.

---

## ✨ Features

* 3D endless runner gameplay
* Smooth left and right movement
* Obstacle avoidance mechanics
* Coin collection system
* Real-time score tracking
* Distance progression
* Collision-based game over
* Endless survival challenge
* Responsive controls

---

## 🕹️ Controls

| Action     | Key             |
| ---------- | --------------- |
| Move Left  | A / Left Arrow  |
| Move Right | D / Right Arrow |

## 🎯 Objective

The goal is to:

* Avoid obstacles
* Collect coins
* Survive as long as possible
